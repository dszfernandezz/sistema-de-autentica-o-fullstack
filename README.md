# Sistema de Autenticação FullStack

![Status](https://img.shields.io/badge/status-em%20desenvolvimento-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)

Este projeto consiste em um sistema completo de autenticação de usuários, abrangendo tanto o frontend quanto o backend. Desenvolvido para oferecer uma experiência segura e fluida de cadastro e login, o sistema implementa validações em tempo real, verificação assíncrona de dados e autenticação baseada em tokens JWT. A aplicação foi construída utilizando React com TypeScript no lado do cliente, proporcionando uma interface moderna e responsiva graças ao Tailwind CSS, enquanto o servidor roda em Node.js com Express, conectando-se a um banco de dados relacional PostgreSQL. Toda a infraestrutura é orquestrada com Docker e Docker Compose, garantindo facilidade de implantação e consistência entre ambientes. No frontend, formulários são gerenciados com Zod e hooks customizados, permitindo validações instantâneas de campos como CPF, e‑mail, telefone e senha, além de consultas assíncronas ao backend para verificar disponibilidade de e‑mail e telefone antes do envio. O fluxo de login conta com uma etapa de pré‑autenticação que só revela o campo de senha após a confirmação da existência do usuário, reforçando a segurança da aplicação. No backend, senhas são armazenadas com hash bcrypt e a comunicação é protegida por CORS e variáveis de ambiente. O resultado é uma base sólida para qualquer projeto que necessite de um módulo de autenticação robusto, moderno e facilmente extensível.
